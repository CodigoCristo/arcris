# ArcriS

1. Descarga la ISO de Arch Linux Oficial.
2. Descargar el Script y ejecutalo.

> Método 1
```
loadkeys es
curl -L is.gd/arcris > arcris ; sh arcris
```

> Método 2
```
loadkeys es
pacman -Sy wget --noconfirm
wget is.gd/arcris ; sh arcris
```
